Programmhilfe

Dieses Tool unterstützt dich beim Erfassen und Exportieren von Getränkedaten.

Tastenkombinationen

  • Enter → Position hinzufügen (im Kisten/Flaschenfeld)
  • Ctrl+S → lokal speichern
  • Ctrl+E → Excel exportieren
  • Ctrl+P → Druckansicht öffnen
  • Ctrl+Z → letzte Entfernung rückgängig
  • Ctrl+M → Mail schicken
  • Ctrl+, → CSV exportieren
  • Ctrl+B → Barcode scannen

Favoriten

Produkte mit dem Stern markieren → erscheinen im Favoriten-Dropdown.

Import/Export

  • CSV → Daten als CSV exportieren oder importieren
  • Excel → Tabellenblatt herunterladen
  • Drucken → Exportiert die Liste als PDF

Tipps zur Nutzung

  • Kommentare für Sonderfälle eintragen (z. B. beschädigt, Sonderlieferung, Personalverzehr)
  • Externe Adressen erscheinen nur, wenn „Extern“ gewählt ist
  • Gibt es das Getränk nicht? Dann wähle die letzte Position in der Getränkeauswahl aus und Folge den Anweisungen! Dann kann ich die Liste entsprechend anpassen.
  • Der Strichlistenmodus lässt dich einzelne Flaschen zählen. In der Auswertung wird alles Summiert.
  • Ein Klick auf den Tabellenkopf sortiert die Liste Auf- oder Absteigend.
